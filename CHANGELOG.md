# Changelog

All notable changes to this plugin are documented in this file.

## 1.0.0 - 2026-07-01

- Forked the original Zoom Codex plugin into `Zoom Revenue Accelerator`.
- Removed the copied non-ZRA commands, skills, and reviewer agents.
- Added ZRA MCP skills for conversation review, deal review, pipeline triage, coaching agenda, follow-up packages, and request routing.
- Removed the old team-scope workflows tied to the previous MCP surface.
- Added ZRA slash commands for setup, debugging, and sales workflows.
- Replaced plugin metadata with Zoom Revenue Accelerator-specific descriptions and version `1.0.0`.
- Added the Zoom Revenue Accelerator app connector ID.
- Updated skills, commands, and references for the live 15-tool ZRA MCP surface from 2026-07-01.
- Added customer context, transcript evidence, and team/user scope skills.
- Added indicator review command and skill coverage for the current ZRA MCP tool surface.
