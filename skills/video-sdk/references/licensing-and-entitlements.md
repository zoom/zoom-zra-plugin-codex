# Video SDK Licensing and Account Prereqs

Common forum questions:

- “How do I assign a Video SDK license to a user?”
- “Why does Video SDK feature X not work on this account?”

## What to Ask

- Is the account enabled for Video SDK?
- Which plan/entitlements are enabled for the account?
- Are they using UI Toolkit features that require paid add-ons (recording, phone, captions, etc.)?

## Admin/Account Guidance

When the ask is “assign license” or “enable feature”, the answer is typically:

- confirm the account/admin configuration in Zoom portal / Marketplace
- confirm the SDK credentials are from the correct account

(This is often not a code change, it’s configuration.)
