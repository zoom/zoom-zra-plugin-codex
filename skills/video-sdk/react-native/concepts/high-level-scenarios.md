# High-Level Scenarios

## 1. Branded mobile collaboration app

- Build custom session UI/UX with your own controls.
- Use chat + command channel for in-session interactions.

## 2. Telehealth/support app

- Join secured sessions from mobile clients.
- Apply strict camera/mic permission and consent handling.

## 3. Creator/live event companion

- Audience or host clients join custom sessions.
- Enable selective features: transcription, recording, share, moderation.

## 4. Enterprise field collaboration

- Operators join task sessions on mobile.
- Use helper modules for audio/video controls and remote assistance patterns.
